About this project:

This project was created by a Software Engineering student, who is currently studying at Facultad de Ingenieria (FING). The main propuse of this page is to show Python skills, by using Django, also a little bit of HTML and CSS is shown here. The functionalities are:

  1. Creation of an appointment object, whose parameters are day, time, and service. (You have be authenticated and handles if all three parameters were selected before)
  2. List the appointments made by the user.
  3. Updating an existent appointment, where you can change all parameters if desired.
  4. Deleting an existent appointment.

You can try it now! Hope you like it.
